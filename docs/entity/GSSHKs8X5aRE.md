## Notes
This chamber opera seems to have been specifically composed for the _Journal_.

It is divided in five sections

1. **Air** (Gilette)
2. **Duo** (Gilette, Gaspardeau)
3. **Romance** (Julie)
4. **Couplet** (Gaspardeau)
5. **Couplets** (Julie, Jacques, Gilette)
6. **Ensemble** (Julie, Jacques, Gilette)

The full libretto is available from [page 140](https://www.digitale-sammlungen.de/en/view/bsb10360013?page=178).
