# Trésor des demoiselles

Knowledge base of 19th century female periodicals, with a particular focus on music.

[Repository website](https://nicholascorniaorpheus.github.io/tresor-des-demoiselles/) build with [Mkdocs Material](https://squidfunk.github.io/mkdocs-material/) deployed via GitHub Pages

The repository has been published and presented by the authors during the Music at Home conference 2026 in Vienna.
